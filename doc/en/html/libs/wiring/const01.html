Fonte ("Power")/Terra("Ground")

Biblioteca: Conectar (Wiring)
Introdução: 2.7.0
Aparência:

Comportamento

Emitirá um único valor pela conexão. Para um elemento do tipo Fonte, indicado por um triângulo, esse valor será igual a 1 (ou, se o atributo de Bits de Dados for maior do que 1, todos os valores iguais a 1). Para um elemento do tipo Terra, indicado por uma seta com três linhas paralelas de tamanho decrescente, esse valor será 0 (ou, se o atributo de Bits de Dados for maior do que 1, todos os valores iguais a zero).

Essa mesma funcionalidade pode ser alcançada usando o componente Constante que é mais versátil. A única razão em preferir esses itens é que eles se referem a símbolos eletrônicos padronizados.

Pinos

Há apenas um pino, uma saída cuja largura de bits será especificada por seu atributo Bits de Dados (Data Bits). O componente emitirá sempre o mesmo valor por esse pino: para um elemento do tipo Terra, a saída terá todos os bits iguais a 0; e para um elemento do tipo Fonte, a saída terá todos os bits iguais a 1.

Atributos

Quando o componente for selecionado ou estiver sendo acrescentado, Alt-0 até Alt-9 alterarão seu atributo Bits de Dados (Data Bits), e as setas alterarão seu atributo de posição (Facing).

Posição (Facing)
A direção preferencial da seta relativa à localização do pino.
Bits de Dados (Data Bits)
A largura de bits do valor a ser colocado na conexão.
Valor
O valor, em hexadecimal, que será emitido pelo componente. O número de bits usados para especificar o valor não poderá exceder a largura em bits do componente.
