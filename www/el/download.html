Λαμβάνοντας το Logisim

Το Logisim είναι ελεύθερο λογισμικό, διαθέσιμο υπό τους όρους του GNU General Public License, version 2. Θα πρέπει να εκτελείται σε οποιαδήποτε πλατφόρμα που υποστηρίζει Java, version 5 ή μεταγενέστερη έκδοση.

  1. Το Logisim χρειάζεται Java 5 ή μεταγενέστερη έκδοση. Αν δεν τη διαθέτετε στον υπολογιστή σας, η Java είναι διαθέσιμη στο java.sun.com.

  2. Λήψη του Logisim από τη σελίδα Logisim sto SourceForge.net. Έχετε τρεις επιλογές ως προς ποια έκδοση να κάνετε λήψη.

    • Ένα .jar αρχείο - θα πρέπει να εκτελείται σε κάθε πλατφόρμα, αν και μερικές φορές όχι ικανοποιητικά.
    • Ένα .tar.gz αρχείο για MacOS X
    • Ένα .exe αρχείο για Windows

    Αν χρησιμοποιείτε MacOS X ή Windows, η έκδοση που αντιστοιχεί στην πλατφόρμα σας θα εκτελείται καλύτερα.

  3. Για την εκτέλεση του προγράμματος:

    • Με το γενικόc .jar αρχείο: Στις περισσότερες πλατφόρμες, εκκινείτε το αρχείο JAR με διπλό κλικ στο εικονίδιο του. Αν το προηγούμενο δεν είχε αποτέλεσμα, εισάγετε στην γραμμή εντολής της πλατφόρμας σας:

      java -jar logisim-generic-XX.jar

    • Με την έκδοση MacOS X: Πρώτα αποσυμπιέστε το αρχείο .tar.gz που κάνατε λήψη: Αυτό μπορεί να γίνει αυτόματα ή να χρειαστεί να κάνετε διπλό κλικ πάνω του. Τότε απλά κάνετε διπλό κλικ στο εικονίδιο του Logisim για να εκκινήσει. Μπορεί να θέλετε να μετακινήσετε το εικονίδιο στο φάκελο Εφαρμογές (Applications).

    • Με την έκδοση για Windows: δεν υπάρχει αρχείο εγκατάστασης του Logisim. Το αρχείο που κάνατε λήψη είναι και το αρχείο που χρειάζεστε για εκτέλεση. Απλά κάνετε διπλό κλικ στο εικονίδιο του Logisim. Μπορεί να θέλετε να δημιουργήσετε μια συντόμευση στην Επιφάνεια Εργασίας και/ή στο μενού Εκκίνησης για να διευκολύνετε την εκτέλεση του Logisim.

  4. Η τυπική έκδοση του Logisim περιλαμβάνει μια Αγγλική μετάφραση (από τον TRANSLATOR NAME του TRANSLATOR INSTITUTION). Για να χρησιμοποιήσετε την Αγγλική μετάφραση:

    1. Αφού εκκινήσετε το Logisim, πηγαίνετε στο μενού Αρχείο και επιλέξτε Προτιμήσεις... (Σε ένα Macintosh, αυτό θα είναι στο μενού Logisim.)
    2. Στην κορυφή του παραθύρου Προτιμήσεων, επιλέξτε την καρτέλα Διεθνοποίηση.
    3. Από το πτυσσόμενο μενού με την επισήμανση Γλώσσα, επιλέξτε την Αγγλική (English) γλώσσα.

Αν βρείτε το Logisim χρήσιμο, παρακαλώ στείλτε μου τα σχόλια σας!
