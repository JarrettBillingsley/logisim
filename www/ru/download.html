Получение Logisim

Logisim должен работать на любой платформе, поддерживающей Java версии 1.5 или выше.

  1. Logisim требует Java 1.5 или более поздней версии. Если у вас её ещё нет на компьютере, то Java доступна на java.sun.com.

  2. Скачать Logisim со страницы Logisim на SourceForge.net . У вас будет на выбор три версии для загрузки.
    • .jar файл - работает на любой платформе, правда он не всегда удобен.
    • Файл .tar.gz для MacOS
    • Файл .exe для Windows
    Если вы используете Windows или MacOS, я бы рекомендовал использовать версию, специфичную для вашей платформы.

  3. Чтобы запустить программу:
    • Для общего .jar файла: На Windows и MacOS вы скорее всего сможете запустить Logisim двойным щелчком на JAR файле. Если это не работает, или если вы используете Linux или Solaris, то вы можете напечатать"java -jar logisim-XX.jar"В командной строке.

    • Для MacOS X версии: когда загруженный файл .tar.gz распакован (это наверняка произойдёт автоматически), просто дважды щёлкните на значке Logisim, чтобы запустить его. Вы можете поместить значок в папку "Приложения".

    • Для Windows версии: Просто дважды щелкните на значке Logisim. Вы можете создать ярлык на рабочем столе и/или в меню "Пуск", чтобы сделать запуск Logisim проще.

Если Вы находите Logisim полезным, пожалуйста пошлите мне комментарий!
